# Drift

A frontend feature-drift monitoring lab for comparing baseline and live distributions across rolling windows.

## Highlights
- Synthetic Gaussian stream generation with tunable mean and variance shifts.
- Drift statistics: Population Stability Index (PSI) and Jensen-Shannon divergence.
- Streaming mode for trend analysis across sequential windows.
- Visualization of distribution overlap and PSI threshold crossings.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS dashboard UI

## Run
Open `index.html` in your browser.
